Here are some of the key conclusions from the survey:

Authentication is the first step to establish secure communication in vehicular networks. It helps distinguish between legitimate and malicious vehicles.

Proxy vehicle-based authentication schemes are suitable for V2R and R2V communications as it reduces the burden on RSUs. However, they have high overheads for V2V communication.

Authentication schemes without proxy vehicles are better for V2V communication as they have lower overheads compared to proxy-based schemes.

Most existing schemes provide privacy-preservation, message integrity and non-repudiation. Few schemes ensure unlinkability and confidentiality.

No scheme is fully secure against denial of service attacks. Most are secure against impersonation, identity revealing and message modification attacks.

Hybrid approaches that combine different authentication techniques can provide better security. But they often have high computational overhead.

Identity-based signature schemes are efficient options as they generate signatures using vehicle identities. Batch verification further reduces overheads.

Further research is needed to design lightweight authentication schemes that ensure all key security requirements with minimum overheads. Real-world deployment challenges also need attention.

In summary, this survey provides useful insights into various authentication schemes proposed for vehicular networks. It can help researchers in identifying suitable techniques and addressing open challenges. The goal should be designing practical secure authentication solutions for safety-critical vehicular communication systems.
